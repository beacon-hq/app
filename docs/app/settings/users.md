# Users

The Users section manages individual user accounts, profiles, and personal settings within the Beacon feature flag system, providing comprehensive user administration and self-service capabilities.

## Overview

User management encompasses individual user accounts, personal profiles, authentication settings, and user-specific configurations. This section provides both administrative user management capabilities and self-service options for individual users.

![Users Overview](screenshots/users-overview.png)
*Main user management dashboard showing user directory and administration tools*
